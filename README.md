# GitHub Profile Finder

A simple web application that allows users to search GitHub profiles using the GitHub REST API.

## 🚀 Features

- 🔍 Search any GitHub username
- 👤 Display profile picture
- 📛 Display user name
- 👥 Display followers count
- 📂 Display public repositories count
- 🔗 Direct link to the GitHub profile
- ⚡ Fast and responsive interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- GitHub REST API
